# Introduction

# Installing Jenkins locally with Docker

You need Bash as Shell.

The following will create a docker container jenkins-blueocean:

```
$ ./setup.sh
```

# Configure Jenkins

Start [the Jenkins client](http://localhost:8080/).

Follow the instructions.
